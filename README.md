# Tech Blog - Homework # 14
## Project Description 
Online tech blog that allows users to create an account, make posts and comment on other posts. 

### Built With 
- HTML
- CSS 
- Javascript
- Express
- Express-Handlebars
- Express-Sessions
- bcrypt
- dotenv
- MySQL
- dbSeed
- Google Fonts 
- Awesome Icons
